Run Blot locally

Use this guide to set up a local Blot development environment.

1. Install Docker

You will need Docker. Ensure that Docker is installed and running on your machine before proceeding.

2. Clone the repository

Use Git to clone Blotâ€™s source code:

git clone https://github.com/davidmerfield/blot

3. Start the server

In the cloned repository, start the server:

npm start

4. Trust local SSL certificates

Before you start editing code, open these URLs and trust the self-signed certificates:

  • https://localhost/
  • https://cdn.localhost/
  • https://example.localhost/

5. Open the dashboard and example site

After trusting the certificates, open:

  • The dashboard: https://localhost
  • The example site: https://example.localhost

6. Edit the example blog

The example blog folder is inside data:

./data/blogs/blog_$ID
