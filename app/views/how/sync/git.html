Git

You can connect your site's folder to Blot with Git, a version-control system for tracking file changes and collaborating with others. You can switch your site to another sync method at any time.

Setting up Git

When you configure your site with git on the dashboard, Blot will provide a repository URL to clone, plus a username and password. 

git clone https://blot.im/url-to-your-repository.git

Your site is generated from the master branch of your repository. To update your site, commit your changes and push.

Using your phone or tablet

There are a few Git clients for phones and tablets. A common option is Working Copy.

Limits

Pushes are subject to a 1 GB limit. Pushes and pulls must send and receive data within 3 minutes each way. Downloads are throttled to 10 MB/s after the first 100 MB of a clone or fetch.

Using an existing Git repository

You can use an existing Git repository. Set its remote URL to Blot's repository endpoint: 

git remote set-url origin https://blot.im/url-to-yours.git

The next time you push, you will need to use the force flag:

git push --force

Sharing your folder on GitHub

Git supports multiple remote URLs. This means you can use the same repository on Blot and GitHub. If you already use GitHub, you can add Blot's endpoint: 

git remote set-url --add --push origin https://blot.im/your-repo.git

If you already use Blot as your remote, you can add GitHub's endpoint: 

git remote set-url --add --push origin https://github.com/your-repo.git

The next time you push, use the force flag so the changes are accepted by both Blot and GitHub: 

git push --force
