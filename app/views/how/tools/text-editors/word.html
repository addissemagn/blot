Microsoft Word

Use Word documents with Blot

Blot can publish Word documents by converting .docx files to HTML.

1. Prepare your document

Write your content in Word. Use styles for headings, lists, and tables so the converted HTML has clear structure.

2. Save as DOCX

Save the file as .docx. This is the format Blot converts.

3. Convert DOCX to HTML

Blot converts DOCX documents using Pandoc. Upload or sync your file, then review the generated HTML and adjust styles in your template if needed.

4. Publish

After syncing, your Word document is published like other Blot content files.

Notes

  • Use Word styles consistently for better output.
  • Complex layouts may need CSS adjustments in your template.
  • For unsupported formatting, export to HTML manually and edit as needed.
